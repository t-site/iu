# iu

## このプログラムは何

bashで構築された、なんてことない日本語をコンピューターが話すだけのプログラムです。

コマンドはたったひとつ、 $ iu です。

## このプログラムは何のための

実は、コンピュータの使い心地を良くするためです。

マシンをよく働かせていると、彼がなにかいいたげにするからです。

そのまま使い込んでいると、こっちの気分まで悪くなるのですが、

英語では言えませんこんなこと、ロボットと仲のいい日本なら言えることですが、

コンピューターは生きている気がして、なにかしゃべらせたかったのです。

日本人にはこのコマンドを実行してもらうと、なにか、意味を感じるはずです。
